## 🛠️ Instalación

Para empezar, asegúrate de tener Python instalado en tu sistema. Recomendamos Python 3.8 o superior.

### Paso 1: Clona el Repositorio
```bash
git clone https://github.com/tu-usuario/fundamentos-python.git
cd fundamentos-python
```

### Paso 2: Ejecuta el Código
```bash
python fundamentos.py
```

¡Eso es todo! No necesitas dependencias adicionales para los ejemplos básicos.

## 📚 Temas Cubiertos

Este proyecto cubre los siguientes temas fundamentales de Python:

- 🔢 **Variables**: Declaración y uso de variables.
- ➕ **Operaciones Matemáticas Básicas**: Suma, resta, multiplicación, división.
- ⚡ **Potencias**: Uso del operador `**`.
- 🔍 **Operadores de Comparación**: `==`, `!=`, `<`, `>`, etc.
- 🔗 **Operadores Lógicos**: `and`, `or`, `not`.
- 🌊 **Estructuras de Control de Flujo**:
  - 🏁 **Condicionales**: `if`, `elif`, `else`.
  - 🔄 **Bucles**: `for`, `while`.
- 🛠️ **Funciones**: Definición y llamada de funciones.
- 📊 **Estructuras de Datos**:
  - 📝 **Strings**: Manipulación de texto.
  - 📋 **Listas**: Colecciones ordenadas.
  - 📦 **Tuplas**: Colecciones inmutables.
  - 📖 **Diccionarios**: Pares clave-valor.
- 🏗️ **Programación Orientada a Objetos**: Clases y objetos.

## 💻 Ejemplos de Código

Aquí tienes un vistazo rápido al código en `fundamentos.py`:

```python
# Saludo inicial
print("Hola Mundo")

# Variables y operaciones
numero1 = 10
numero2 = 20
resultado = numero1 + numero2
print(f"La suma es: {resultado}")

# Formateo de strings
print("La suma es: " + str(resultado))
print("La suma es: {}".format(resultado))

# Potencias
potencia = numero1 ** numero2
print(f"{numero1} elevado a {numero2} es: {potencia}")
```

**Salida Esperada**:
```
Hola Mundo
La suma es: 30
La suma es: 30
La suma es: 30
10 elevado a 20 es: 100000000000000000000
```

¡Prueba modificando los valores y viendo qué pasa! 🔬
